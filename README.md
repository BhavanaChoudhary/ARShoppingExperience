# AR Virtual Shopping App

An Android application that lets users browse products, view detailed descriptions, try products in Augmented Reality (AR), and make purchases. Built with Kotlin and Material Design components.

---

## Features

- Browse a list of products with images and pricing.
- View detailed product descriptions.
- Try products in your home using Augmented Reality (AR) via Google Scene Viewer.
- Add products to the cart or buy them instantly.
- **User Authentication:** Firebase is used for login and registration (Email/Password).
- Clean and modern UI with Material Design components.
- Responsive layout compatible with multiple screen sizes.

---

## Technologies Used

- **Language:** Kotlin  
- **IDE:** Android Studio  
- **UI Components:** Material Design, ConstraintLayout  
- **Authentication:** Firebase Authentication (Email/Password)  
- **Augmented Reality:** Google Scene Viewer  
- **Version Control:** Git & GitHub
